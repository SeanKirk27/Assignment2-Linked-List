Detailed Description

Implementation of functions for managing/operating on an ordered integer set.

This file provides the implementation for:

  • Creating and deleting ordered sets.
  • Adding, removing, and searching elements.
  • Performing set operations such as union, intersection, and difference.
  • Printing set contents.

◆ addElement()

ReturnValues addElement ( orderedIntSet * s,
int elem )

Adds an integer element to the ordered set.

Precondition
The set s must be valid (non-NULL).
Postcondition
The integer elem is inserted in sorted order if not already present.
  • Traverses the list to find the correct position for elem.
  • If elem already exists, the function returns an error code.
  • Otherwise, elem is inserted, and the set's size is incremented.

Pseudocode:

  1. If s is NULL:
    • Return ALLOCATION_ERROR
  2. Move to the head of s->list
  3. Traverse list while current->next is not NULL:
    • If currentElement equals elem, return NUMBER_ALREADY_IN_SET
    • If currentElement is greater than elem, break loop
  4. Create newData for elem
  5. Insert newData after the current node:
    • If insertion fails, return ALLOCATION_ERROR
  6. Increment s->size
  7. Return NUMBER_ADDED
Parameters
sA pointer to the orderedIntSet.
elemThe integer element to be added.
Returns
NUMBER_ADDED if successful, NUMBER_ALREADY_IN_SET if elem exists, or ALLOCATION_ERROR on failure.

Creates a new ordered integer set.

Precondition
None.
Postcondition
A new ordered set is initialized and ready for operations.
  • Allocates memory for a new orderedIntSet structure.
  • Initializes the set's size to zero and creates an underlying double linked list to manage the set elements.

Pseudocode:

  1. Allocate memory for orderedIntSet
  2. If memory allocation successful, then:
    • Initialize size to 0
    • Create a double-linked list and assign to list
  3. If list creation fails:
    • Print error message
    • Exit program
  4. Return orderedIntSet
Returns
A pointer to the newly created ordered set, or exits the program on allocation failure.

◆ printToStdout()

int printToStdout ( orderedIntSet * s)

Computes and prints the contents of the given ordered set to the standard output.

Precondition
A valid ordered set exists.
Postcondition
The elements of the set are printed in ascending order.
  • Iterates through all elements in the ordered set and prints their values.
  • Elements are printed in the format {e1, e2, e3, ...}.
  • If the set is empty or NULL, prints {}.

Pseudocode:

  1. If s is NULL:
    • Print "{}" and return
  2. Move to head of s->list
  3. Print "{"
  4. Traverse s->list and print each element:
    • If not last element, print ", "
  5. Print "}" and new line
  6. Return
Parameters
s- A pointer to the 'orderedIntSet' to be printed.
Returns
The number of elements in the set.
Note
This function does not modify the structure of the set.

int elem )

Removes an integer element from the ordered set.

Precondition
The set s must be valid (non-NULL).
Postcondition
If elem is present, it is removed and the set's size is decremented.
  • Searches for elem in the set.
  • If found, it removes the corresponding node.
  • If not found, an error code is returned.

Pseudocode:

  1. If s is NULL:
    • Return ALLOCATION_ERROR
  2. Move to the head of s->list
  3. Traverse list using gotoNextNode():
    • If currentData equals elem:
      • Delete current node
      • Decrement s->size
      • Return NUMBER_REMOVED
  4. If element not found:
    • Return NUMBER_NOT_IN_SET
Parameters
sA pointer to the orderedIntSet.
elemThe integer element to be removed.
Returns
NUMBER_REMOVED if elem was removed, NUMBER_NOT_IN_SET if not found, or ALLOCATION_ERROR on failure.

◆ setDifference()

orderedIntSet * setDifference ( orderedIntSet * s1,
orderedIntSet * s2 )

Computes the difference between two ordered sets and returns the result as a new set.

Precondition
Two valid ordered sets exist.
Postcondition
A new ordered set is created containing elements in s1 that are not in s2.
  • The difference operation returns a set containing all elements that are present in s1 but not in s2.
  • The resulting set is sorted, and both input sets remain unchanged.

Pseudocode:

  1. If s1 or s2 is NULL:
    • Return NULL
  2. Create new ordered set differenceResult
  3. Traverse s1->list from head:
    • For each element in s1, traverse s2->list:
      • If element is not found in s2, add it to differenceResult
  4. Return differenceResult
Parameters
s1- A pointer to the first 'orderedIntSet' (the minuend set).
s2- A pointer to the second 'orderedIntSet' (the subtrahend set).
Returns
A pointer to a new 'orderedIntSet' representing the difference of s1 and s2. Returns NULL if s1 is NULL. If s2 is NULL, the result is a copy of s1.
Note
Memory allocation is performed for the new set. The caller is responsible for freeing this memory.

Computes the intersection of two ordered sets.

Precondition
Both sets s1 and s2 must be valid (non-NULL).
Postcondition
Returns a new set containing elements common to both s1 and s2.
  • Iterates over both sets to identify common elements and adds them to a new set.

Pseudocode:

  1. If s1 or s2 is NULL:
    • Return NULL
  2. If s1 equals s2:
    • Return s1
  3. Create new ordered set intersectionResult
  4. Traverse s1->list from head:
    • For each element in s1, traverse s2->list from head:
      • If an element in s1 equals an element in s2:
        • Add element to intersectionResult
  5. Return intersectionResult
Parameters
s1A pointer to the first orderedIntSet.
s2A pointer to the second orderedIntSet.
Returns
A pointer to the resulting set, or NULL on failure.

Computes the union of two ordered sets and returns the result as a new set.

Precondition
Two valid ordered sets exist.
Postcondition
A new ordered set is created containing all unique elements from both input sets.
  • The union operation combines all elements from set s1 and set s2.
  • The resulting set contains only unique elements, maintaining sorted order.
  • Both input sets remain unchanged.

Pseudocode:

  1. If s1 or s2 is NULL:
    • Return NULL
  2. Create new ordered set unionResult
  3. Traverse s1->list from head:
    • Add each element to unionResult
  4. Traverse s2->list from head:
    • Add each element to unionResult
  5. Return unionResult
Parameters
s1- A pointer to the first 'orderedIntSet'.
s2- A pointer to the second 'orderedIntSet'.
Returns
A pointer to a new 'orderedIntSet' representing the union of s1 and s2. Returns NULL if both input sets are NULL.
Note
Memory allocation is performed for the new set. The caller is responsible for freeing this memory.
