Detailed Description

Header file for defining function declaration for managing a double linked list.

Defines the data structures and error codes functions for managing and operating the double linked list.

This header file includes function declarations only, while function definitions are implemented in 'DoubleLinkedListFunctions.c'. Used for functions like to manipulate and operate the double linked list.

Note
data structures ('DoubleLinkedList', 'Node', 'data') and error codes ('llError') used by these functions are defined in 'DoubleLinkedListTypeDefs.h'

◆ createDoubleLinkedList()

DoubleLinkedList * createDoubleLinkedList ( )

Creates a new empty double linked list, consisting only of head and tail.

Precondition
None
Postcondition
Empty double linked list is created
  • Allocates memory for the list and its head node.
  • Initialises the head node with no next or previous nodes.
  • Head points to tail resulting in NULL.
  • Set the current pointer to the head node.

PSEUDOCODE

  1. Allocate memory for list
  2. If successfully allocated, then;
    • Allocate memory for head node.
  3. then if head allocation successful, then;
    • Set next & previous node from head to NULL 4, else
    • free lists memory
    • set list to NULL
  4. return list, end.
Returns
: Pointer to created DoubleLinkedList if successful, Pointer to NULL if it fails.

llError deleteCurrent ( DoubleLinkedList * list)

Deletes the current node from list.

Precondition
valid double linked list exists.
Postcondition
if current node is other than head or tail, the current node is removed from list. Otherwise, an error is returned.

Removes the node currently pointed to. The previous and next nodes are placed together. If the current node is the head or of NULL type, it does nothing and returns an error status.

PSEUDOCODE

  1. If current is head or tail, then;
    • return illegal error
  2. store current (which will be deleted) in a temperary varable
  3. if next is null, then;
    • set current's previous ad next to NULL
  4. else set the following;
    • current previous next to current next
    • curent next previous to current previous
  5. set current to current's previous
  6. free the temp varable
Parameters
list- A pointer to the 'DoubleLinkedList', thats where the current node is deleted from.
Returns
'llError' status code
Note
cannot delete the head node
Warning
after deleting a node, the current pointer is updated to the previous node.

◆ deleteDoubleLinkedList()

void * deleteDoubleLinkedList ( DoubleLinkedList * list)

Deletes the double linked list (including all its elements) and frees all associated memory.

Precondition
valid double linked list exists
Postcondition
The entire list is deleted.

This deletes all nodes in the 'DoubleLinkedList' and frees the memory allocated for the list structure itself. Iterates through all nodes and frees them one by one.

PSEUDOCODE

  1. dont do anything if list is NULL
  2. Set current node to head
  3. while loop for it to get to the end of the loop (Next doesnt equal NULL).
    1. Set current node to head
    2. Set next node to next next node
    3. free up the next node.
  4. free head and list after ending while loop
Parameters
list- A pointer to the 'DoubleLinkedList' thats to be deleted
Note
If the list pointer is 'NULL', the function does nothing.
Warning
once called, all pointer referencing nodes will become invalid.
